# Dotfiles RICE Installer

This script installs necessary dependencies, clones the RICE into your HOME directory, creates a secure backup of your files, and copies the new files to your computer. 

## Installation

1. Clone the repository:

git clone https://github.com/Irmb7108/dotfiles.git


2. Change into the dotfiles directory:

cd dotfiles


3. Run the installer script:

./RiceInstaller


4. Follow the prompts to install dependencies, clone the RICE, and copy files to your system.

## Notes

- My dotfiles do not modify any of your system configurations.
- You will need to enter your root password to install missing dependencies or switch to the zsh shell if it's not your default.
- This script cannot break your system as it only copies files from my repository to your HOME directory.
