# RiceInstaller

RiceInstaller is a Bash script that automates the process of configuring a Linux system with my personal dotfiles. The script downloads and installs a number of command-line tools, applications, and plugins, and configures them to work together seamlessly.

## Installation

To install RiceInstaller, simply clone this repository and run the `RiceInstaller` script:

git clone https://github.com/Irmb7108/dotfiles.git
cd dotfiles
./RiceInstaller


## Usage

Once installed, RiceInstaller will configure your Linux system with my personal dotfiles. This includes a number of custom settings, aliases, functions, and plugins that I find helpful for my workflow. You can modify these dotfiles to suit your own preferences by editing the files in the `~/dotfiles` directory.
